# openclaw-feishu

A Codex skill for configuring, auditing, and troubleshooting the OpenClaw Feishu/Lark channel.

Source docs: https://docs.openclaw.ai/zh-CN/channels/feishu

## Structure

- `SKILL.md`: trigger description and execution workflow
- `references/setup.md`: first-time setup, permissions, event subscription, config, and test flow
- `references/operations.md`: access control, ACP, Drive comments, advanced config, and troubleshooting
